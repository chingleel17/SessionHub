# 安全性政策

## 回報安全漏洞

請不要在公開 Issue 或討論區張貼安全漏洞、API key、Access Token、session 內容或其他敏感資料。

優先使用 GitHub Security Advisories 的 **Report a vulnerability** 私下回報。如果該功能不可用，請透過 repository owner 的 GitHub 個人頁面聯繫維護者，並提供：

- 受影響的版本或 commit
- 重現步驟與必要的最小範例
- 可能的影響與建議修正方式
- 是否已在其他地方公開

## 支援範圍

目前只支援最新 `main` 與最新公開版本。安全修正完成後，維護者會在適當時機發布版本與變更說明。
